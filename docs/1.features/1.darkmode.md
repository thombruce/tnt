# Darkmode

TNT pages feature light and dark themes. The default layout also includes a darkmode toggle so that users can change the theme on the fly. Try it here...

Click the icon: :tnt-darkmode-toggle

TNT features three theme modes:

1. Light mode
2. Dark mode
3. System mode (default)

The page will match the user's system preferences by default. Pressing the toggle will switch the current preference to either light or dark mode and save this choice in the user's browser.
