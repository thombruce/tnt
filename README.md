## Usage

_Configure NUXT_APP_BASE_URL to the name of your repo (e.g. /tnt/) in your environment if you want to deploy to GitHub Pages from a project repo._

### Installation

```sh
yarn add -D @thombruce/tnt
```

### Updating

```sh
yarn upgrade @thombruce/tnt
```
