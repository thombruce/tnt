# Installation

::lead
Step 1.
::

```sh
yarn add @thombruce/tnt-content
```

::lead
Step 2.
::

```ts
export default defineNuxtConfig({
  extends: [
    '@thombruce/tnt-content'
  ],
})
```

::lead
Step 3.
::

**That's it!**

You're ready to create your first markdown document in your `/content` directory. _Happy writing!_
